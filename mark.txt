# Оформление заказа

**Цель: оформить доставку пиццы на сайте**

1.	Пользователь переходит по ссылке сайта компании-поставщика пиццы
2.	Система отображает стартовую страницу с формой авторизации
3.	Пользователь осуществляет вход в систему с помощью электронной почты


_Альтернативно_

3.	Пользователь осуществляет вход в систему с помощью аккаунта соцсети
Альтернативно
4. Пользователь регистрируется в системе как новый

**Система отображает основной функциональный экран с перечнем пицц для заказа**
